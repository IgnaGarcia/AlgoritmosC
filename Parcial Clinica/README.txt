Programa que simula una clinica donde se es socio o es
cliente privado, este puede elegir entre diferentes
especialidades para una consulta, donde cada una tiene 
un propio costo.

Al final del programa se informa cuantos turnos para 
socios se dieron, cuantos para privados, el total de turnos
lo recaudado en cada especialidad, lo recaudado total y el
paciente de menor edad.
